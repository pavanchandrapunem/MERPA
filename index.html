About Our Project

The Microsoft Edge Reward Points Automation project is a real-time web automation tool built using HTML, CSS, and JavaScript. The primary goal of this project is to automate the collection of Microsoft Reward Points within the Microsoft Edge browser. The webpage created for this project automates the process of gathering reward points by mimicking user activity and performing various tasks like searching, browsing, and interacting with the browser, thus helping to accumulate points without manual input.

Over the span of two months, this project has proven its effectiveness, allowing the user to redeem 8 Flipkart Gift Cards, each valued at ₹100, showcasing the successful automation of the reward system.

  • Automation with JavaScript: The core functionality is powered by JavaScript, which drives the automation process, simulating user behavior to perform tasks that earn Microsoft Reward Points.
  • Responsive UI Design: Developed with HTML and CSS, the webpage is designed to be user-friendly and responsive, ensuring that it runs smoothly across devices.
  • Real-Time Automation: The system works continuously in real-time, allowing points to accumulate passively over time.
  • Rewards: Within two months, the project resulted in the redemption of ₹800 Flipkart Gift Cards, demonstrating the potential financial benefits of automating reward collection.

This project showcases the power of web technologies in creating efficient automation systems that can be utilized for practical, real-world applications like rewards and earning.
